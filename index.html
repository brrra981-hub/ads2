Mantenimiento de sitios web

Mantener tu Web actualizada es sumamente importante para la seguridad de tu datos. Por eso, te recomendamos tener tu web siempre actualizada.

Accesibilidad web

Evaluamos la accesibilidad de tu web. Basándonos en las pautas de accesibilidad para el Contenido Web 2.1 (WCAG 2.1), rediseñamos, y realizamos pruebas de calidad. Nuestra cadena de valor es generar trabajo para las personas con discapacidad, y que los contenidos sean inclusivos.
